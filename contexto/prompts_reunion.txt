📹 VÍDEO 1: "CLAUDE CODE 2026: Curso Completo en Español"
Canal: Benjamín Cordero

Qué es Claude Code:

NO es el chat de Claude.ai. Es una herramienta de terminal (CLI) que corre en tu ordenador.
Se instala con: npm install -g @anthropic-ai/claude-code
Se abre con el comando: claude en la terminal, dentro de la carpeta del proyecto
Lo más importante que debes retener:

CLAUDE.md — El archivo más importante. Es donde defines el contexto del proyecto para que Claude lo lea al inicio de cada sesión. Sin él, pierdes contexto y gastas tokens.
Flujo de trabajo correcto: Explorar → Planificar → Codificar (nunca saltarse el paso de planificación)
Modo /plan — Antes de ejecutar, pídele que planifique. Te ahorra errores caros.
Permisos — Claude Code puede leer y editar archivos, ejecutar comandos en terminal, y navegar por el sistema. Al inicio te pide confirmación para cada acción nueva.
Lo que NO hacer:

No abrir Claude Code sin tener el CLAUDE.md preparado
No darle un prompt gigante de golpe. Ve paso a paso.
No dejar sesiones abiertas largas sin guardar el contexto en NotebookLM
📹 VÍDEO 2: "Claude Code + NotebookLM = MEMORIA INFINITA" ⭐ EL MÁS IMPORTANTE
Concepto: Conecté Claude con NotebookLM (Contexto Infinito Sin Gastar Tokens)

El problema que resuelve: Claude Code "olvida" todo al cerrar la sesión. Cada vez que abres una sesión nueva, tienes que volver a explicar el proyecto → gastas tokens → tardas más.

La solución (cómo funciona):

NotebookLM (de Google) actúa como la "memoria externa" de Claude
Subes documentos al Notebook (PDF, Markdown, URLs, vídeos de YouTube)
Configuras el MCP de NotebookLM para que Claude Code pueda consultarlo directamente
Al inicio de cada sesión, Claude consulta el Notebook en lugar de pedirte que re-expliques todo
Flujo con memoria infinita:

Sesión anterior termina → Claude guarda resumen en NotebookLM
↓
Nueva sesión empieza → Claude consulta NotebookLM
↓
Claude sabe exactamente en qué punto estabas, sin reexplicar nada
↓
Ahorras tokens + tiempo
Lo que necesitas instalar:

Paquete MCP: notebooklm-mcp-cli (del repositorio jacob-bd en GitHub)
Autenticación con tu cuenta de Google (usa 
burguardreams@gmail.com
)
Un Notebook en NotebookLM dedicado a TonExecutive.ai
Tip clave: Al final de cada sesión de trabajo, escribe en Claude Code: "Guarda un resumen de lo que hemos hecho hoy en NotebookLM"

📹 VÍDEO 3: "Creé un equipo de Marketing con Agentes de IA en Claude Code"
Link: https://www.youtube.com/live/ocm6T19g6iM

Qué aprenderás: Claude Code puede orquestar múltiples "agentes" especializados que trabajan en paralelo o en secuencia.

Estructura de un equipo de marketing con IA:

Agente	Rol	Qué hace
Estratega	Orquestador	Coordina a todos los demás
Copywriter	Contenido	Escribe posts, emails, scripts
SEO Analyst	Optimización	Keywords, meta tags, estructura
Social Media	RRSS	Adapta contenido por plataforma
Analytics	Datos	Mide resultados e identifica mejoras
Aplicación directa a TonExecutive.ai:

Agente de contenido: genera posts de LinkedIn semanales para Ton
Agente de benchmark: analiza webs de competidores automáticamente
Agente de leads: procesa y cualifica los leads que entran
Cómo se crean los agentes: En el archivo CLAUDE.md defines los "Skills" — básicamente, instrucciones específicas para cada rol. Claude los activa con comandos como /skill copywriter.
