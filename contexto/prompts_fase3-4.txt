PROMPT F3-A — Integración Whisper (transcripción de voz)
En src/pages/AgentCAIO.tsx, añade un botón de micrófono junto al input de chat.
Al hacer clic:
1. Graba audio del navegador (MediaRecorder API)
2. Al parar, envía el blob de audio a POST /api/whisper en server.ts
3. El endpoint recibe el audio, lo envía a OpenAI Whisper API y devuelve el texto
4. El texto transcrito aparece automáticamente en el input de chat
En server.ts:
- Añade POST /api/whisper
- Usa FormData para enviar el audio a https://api.openai.com/v1/audio/transcriptions
- Model: "whisper-1", language: "es"
- Requiere OPENAI_API_KEY en .env
UI: botón rojo pulsante mientras graba, icono de micrófono (Lucide Mic / MicOff).
Si el navegador no soporta MediaRecorder, muestra tooltip "Tu navegador no soporta grabación".
